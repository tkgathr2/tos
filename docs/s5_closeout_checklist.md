# S-5 Closeout 準備完了チェックリスト

## 概要

本チェックリストは S-5 closeout を完了するために必要な項目を網羅しています。
すべての項目にチェックが入った状態で、S-5 closeout 完了とみなされます。

## 使用方法

1. 各項目を順番に確認する
2. 確認が完了したら `[ ]` を `[x]` に変更する
3. すべての項目が `[x]` になったら closeout 完了
4. 完了後、本チェックリストを Git にコミットする

## Closeout 完了判定基準

以下がすべて満たされた場合、S-5 closeout は完了です：

1. 本チェックリストのすべての項目が `[x]` である
2. docs/s5_closeout.md が存在し、宣言文が記載されている
3. GitHub main ブランチに最新の変更が push されている
4. 運用者（または開発者）による最終確認が完了している

## Closeout 完了後

### 許可される作業

- ドキュメントの軽微な修正（誤字脱字、説明の明確化）
- コメントの追加
- 既存機能のテスト追加
- 運用中の問題報告

### 禁止される作業

- 新機能の追加
- 仕様変更
- 設定スキーマの変更
- 破壊的変更
- orchestrator_v0_3.py の改変
- config_v0_3.json のスキーマ変更
