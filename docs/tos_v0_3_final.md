## ステータス定義

**TOS v0.3 は以下の状態にあります：**

1. **完成**: すべての機能が実装・検証・監査済み
2. **封印**: これ以上の成長・拡張は行わない
3. **運用のみ**: 許可される操作は運用に限定

TOS v0.3 は「育てる対象」ではなく「使う対象」です。

## 以後の禁止操作

以下の操作は**禁止**されます：

| 禁止操作 | 理由 |
|---------|------|
| 実装変更 | 完成・封印状態に反する |
| 仕様変更 | 安定性を損なう |
| 拡張検討 | S-7 不要で確定済み |
| 新機能追加 | 正史コンテキストに違反 |
| 設定スキーマ変更 | 後方互換性を破壊する |
| 依存関係の変更 | 動作保証ができなくなる |
| S-5 ファイルの変更 | 履歴フェーズとして凍結済み |
| 改修依頼の受付 | 封印状態では対応不可 |
| 独自拡張の追加 | ドキュメントにない操作は禁止 |

TOS v0.3 は完成品として最終確定されており、いかなる例外も認められません。
